\chapter*{CONCLUSIONES}
\addcontentsline{toc}{chapter}{CONCLUSIONES}
\markboth{CONCLUSIONES}{CONCLUSIONES}

A partir del desarrollo de la presente investigación se logra arribar a las siguientes conclusiones generales:

\begin{enumerate}
	\item La toma de decisiones en contextos complejos con múltiples criterios constituye un proceso fundamental en diversos ámbitos profesionales, el cual se ve limitado por los métodos manuales tradicionales debido a la subjetividad, la dificultad para manejar grandes volúmenes de datos y la incapacidad para procesar información imprecisa.
	
	\item El análisis de los métodos MCDM permitió establecer que herramientas como AHP, TOPSIS, ELECTRE y PROMETHEE, proporcionan un marco conceptual sólido para abordar problemas complejos de decisión, facilitando la evaluación de alternativas bajo múltiples criterios.
	
	\item El uso de Python como lenguaje de programación, junto con Visual Studio Code y herramientas CASE como Visual Paradigm, permitió desarrollar una solución que cumple con los requisitos establecidos de portabilidad y eficiencia.
	
	\item La metodología de desarrollo ágil XP demostró ser efectiva para este proyecto, permitiendo la implementación iterativa e incremental del software mediante 17 historias de usuario organizadas en 4 iteraciones, lo que facilitó la adaptación a los cambios y garantizó la calidad del producto final.
	
	\item La arquitectura de N capas y el patrón MVC implementados proporcionaron una estructura robusta y mantenible para el software, permitiendo una clara separación de responsabilidades y facilitando la extensibilidad del sistema para incorporar nuevos métodos MCDM.
	
	\item Las pruebas de software realizadas demostraron que la implementación satisface las necesidades del cliente, con cero no conformidades después de cinco iteraciones de pruebas unitarias y de aceptación.
	
	\item La validación mediante Pruebas de Veracidad confirmó la exactitud del software con una concordancia del 100\% con los resultados publicados en literatura científica, validando la correcta implementación de los métodos MCDM.
	
	\item La solución desarrollada constituye una contribución significativa al campo de la ingeniería informática, proporcionando una herramienta tecnológica que optimiza los procesos de toma de decisiones multicriterio, reduce costos y tiempos de análisis, y ofrece una plataforma flexible y escalable para futuras investigaciones.
\end{enumerate}
