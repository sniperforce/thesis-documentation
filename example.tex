\abstract{Esta investigación se centra en el desarrollo de un software para el apoyo en la toma de decisiones basado en métodos de decision multicriterio, con el objetivo de minimizar la subjetividad, mejorar la eficiencia del análisis, permitiendo el manejo de grandes volúmenes de datos y el análisis de sensibilidad.  El estudio surge de la necesidad de contar con una plataforma integrada que permita aplicar y comparar resultados de diferentes métodos de toma de decisión multicriterio de manera eficiente, superando las limitaciones de las herramientas existentes que generalmente implementan un solo método o carecen de capacidades de comparación robustas. El software desarrollado permite la entrada de datos, la ponderación de criterios, la aplicación de métodos MCDM (como AHP, TOPSIS, PROMETHEE y ELECTRE) y la visualización de resultados mediante una interfaz gráfica intuitiva. La investigación se basa en una metodología ágil, específicamente Extreme Programming (XP). Para la validación de OptiChoice se realizaron pruebas de software a nivel de pruebas unitarias y de aceptación, con el objetivo de validar el cumplimiento de los requisitos y garantizar la calidad del sistema. Además se adicionan  pruebas de veracidad con el objetivo de verificar la precisión y exactitud de los métodos implementados, a partir de comparación con los resultados de artículos científicos. Las conclusiones destacan que el software no solo optimiza los procesos de toma de decisiones, sino que también ofrece una plataforma flexible y escalable para futuras investigaciones.\\}
